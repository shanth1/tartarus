# knowledge
Глобальная база знаний (паттерн LLM Wiki, Karpathy).
- inbox/ — сюда пишут агенты и Денис. Сырые заметки, один файл — одна тема.
- raw/ — обработанные источники, иммутабельно. Раскладка raw/YYYY/MM/.
- wiki/ — курируемые страницы. Пишет только Архивариус. Читают все агенты с доступом.
Векторный слой: builtin memory OpenClaw индексирует wiki/ через memorySearch.extraPaths (sqlite-vec + FTS5, гибрид).
